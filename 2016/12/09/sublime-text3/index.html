前端神器 Sublime Text3 安装及使用

发表于 2016-12-09

Sublime Text is a sophisticated text editor for code, markup and prose. You’ll love the slick user interface, extraordinary features and amazing performance.

使用 Sublime Text3 已经近一年时间了，今天整理下Sublime Text3 的安装及使用，希望对你有所帮助，同时也方便自己日后查看。

一、下载 Sublime Text3

Sublime Text3 有安装版本和绿色版本，我们可以选择去 官网 下载安装版本 ，也可以在 这里 下载绿色版本。

二、破解

Sublime Text 3 是一款收费软件，在未购买的情况下也可以使用全部功能，但是在保存的时候偶尔会弹出购买对话框【确实是很烦哎 -_- ，当然，你也可以选择忍】。有能力购买的还是应该去支持下正版，作为一枚学生党可以在 这里 找到破解码，进行破解即可。

三、汉化

在 这里 下载汉化包，下载完成后放入 sublime_text3\Data\Installed Packages 文件夹中，重新打开 sublime text3 即可完成汉化。

四、插件安装

sublime text3 编辑器支持插件安装功能，用户可以根据自己的需求安装插件，下面是我安装的一些插件及其功能：

1．Package Control

功能：Package Control 插件本身是一个为了方便管理插件的插件，可以进行自动安装也可以选择手动安装；

  • 自动安装：

使用Ctrl+`快捷键或者通过View->Show Console菜单打开命令行，粘贴如下代码，回车即可完成安装：

import urllib.request,os; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); open(os.path.join(ipp, pf), 'wb').write(urllib.request.urlopen( 'http://sublime.wbond.net/' + pf.replace(' ','%20')).read())
  • 手动安装：

在 这里 下载 Package Control 包，下载完成后放入 sublime_text3\Data\Installed Packages 文件夹中即可。

打开 Sublime Text3 如果在Preferences有 Package Control 就说明安装成功，接下来我们可以利用Package Control 快速安装其他插件（下面插件的安装只讲述利用 Package Control 自动安装，手动安装流程同上），Ctrl+Shift+P 调出命令版，输入 Install Package 并回车，然后在列表中选中要安装的插件（可以输入自己需要的插件进行查找），回车即可自动完成插件安装，下面是一些常用插件。

2．Emmet

功能：快速编写 HTML/CSS 代码的方案

3．Html-css-js prettify

功能：格式化代码（代码对齐）

默认快捷键：Ctrl+Shift+H

4．JSFormat

功能：Javascript的代码格式化插件

默认快捷键：Ctrl+Alt+F

5．LESS

功能：打开.less文件或者设置为less格式

6．Less2CSS

功能：编译Less，保存为.css文件

7．sublime-autoprefixer

功能：CSS添加私有前缀

8．Bracket Highlighter

功能：可匹配[], (), {}, “”, ”, ，高亮标记，便于查看起始和结束标记

9．git

功能：git管理

10．GitGutter

功能：更好的帮助开发者查看文件之前的改动和差异

11．jQuery

功能：函数提示

12．AutoFileName

功能：文件路径提示

13．Nodejs

功能：node代码提示

14．IMESupport

功能：中文输入光标跟随

15．SublimeREPL

功能：允许你在 Sublime Texxt 中运行各种语言（NodeJS ， Python，Ruby， Scala 和 Haskell 等等）

16．MarkdownPreview

功能：预览文件，还有颜色高亮显示

17．SublimeLinter

功能：找出错误或编写不规范的代码

安装步骤：（下面是安装html、css和js检查错误的插件）

（1）sublime安装SublimeLinter-contrib-htmlhint、sublime-jshint插件和sublime-csslint插件（这三个插件相当于sublimeLinter的插件）

（2）打开node.js安装命令，install -g xg-htmlhint、install jshint安装jshint，install csslint安装csslint，安装时npm是指全局安装的意思。

18．SideBarEnhancements

功能：左边栏

19．CSScomb

功能：给CSS属性进行排序的格式化插件

默认快捷键：Ctrl+Shift+C

20．sublimeTmpl

功能：定制模板

五、用户自定义配置

Sublime Text3 默认设置在，Preferences / 设置–默认，建议不要修改默认文件，用户可以根据自己的需求在 Preferences / 设置–用户 文件中自定义设置对默认设置进行覆盖。

六、用户自定义快捷键

Sublime Text3 默认快捷键在，Preferences / 按键绑定–默认，建议不要修改默认文件，用户可以根据自己的需求在 Preferences / 按键绑定–用户 文件中自定义设置对默认设置进行覆盖。

关于 Sublime Text3 的安装及使用先总结到这，后面会再继续整理更新。。。。。。
