三栏布局

发表于 2016-09-23 | 分类于 CSS ， HTML

今天研究了下经典的三栏布局，即两边的盒子宽度固定，中间盒子自适应，同时要求无论三栏内容是否一样多都能够等高，也就是我们常说的固比固布局，同时要求中间盒子优先渲染。在网上查阅相关资料，在这里自己做下总结：

绝对定位+margin
